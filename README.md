## Features

### Design
- Dark & Light theme toggle with smooth ripple effect.
- Fully responsive - work on mobile as well as on tabs, and desktops.
- Custom colors using CSS variables.
- Floating orbs.
- Scrolling marquee.

### Custom cursor
- Hand-drawn cursor.
- Grows on hover and interaction on elements.
- Tilts on clicks.
- Theme-aware fully responsive while theme switches.

### Interations and Animation
- Scroll reveal animation .
- Fade-in animation for hero contents.
- 3D card tilt on projects card with mouse tracking.
- Staggered skill card animation.
- Animated number counters in hero stats.
- Active nav link it get highlighted on scroll position.

### Sound-Effects(web API - no mp3 files)
- Soft click sound on all buttons.
- Two-ton sound effect on toggle button.
- whoosh sound on nav links.

### Responsive
- Hamburger menu for mobile with ESC key + outside click to close.
- Grid layout.
- `prefers-reduced-motion` for accessibility

## How To Run locally

No build step needed. Just open it

```bash
git clone https://github.com/baishnabikumari/My-Portfolio---For-hctg.git
cd My-Portfolio---For-hctg
open index.html
```

or use Live Server in VS code.
